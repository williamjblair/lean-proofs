# Historical `10^166` CF checkpoint

This directory preserves the first exact continued-fraction extension of the
six odd rows, from `10^120` to `10^166`.  Its generators, artifacts, tests,
and audits remain reproducible historical evidence.

It is not the current frontier.  The canonical checked-in band now reaches
`10^1000`; use `../agent_cf_tail_e1000/` for its independent verifier and
audit, and the repository's `ErdosProblems/Erdos686CFTailBand*.lean` modules
for the current kernel certificates.
