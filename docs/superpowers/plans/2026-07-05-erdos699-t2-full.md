# Erdős 699 T2 Full Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Complete the Lean proof of T2: for `i = 2`, every `2 < j` with `2 * j <= n` has a relevant common prime divisor of `C(n,2)` and `C(n,j)`.

**Architecture:** Reuse `t2_collapse_of_no_common` to reduce a hypothetical obstruction to `n = 2 * j` with `j` odd. Pick a prime divisor `p` of `2 * j - 1`; prove it is odd, prove the units digit of `2 * j` is `1` mod `p`, then use Lucas to show `p` divides both `C(2*j,2)` and `C(2*j,j)`.

**Tech Stack:** Lean 4.29.1, Mathlib 4.29.1, exact Python criterion tests, repository shell gates.
