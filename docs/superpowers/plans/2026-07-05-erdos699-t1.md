- [ ] **Step 1: Add predicate**

Add:

```lean
def commonPrimeDivisor (n i j p : ℕ) : Prop :=
  p.Prime ∧ i ≤ p ∧ p ∣ Nat.choose n i ∧ p ∣ Nat.choose n j
```

- [ ] **Step 2: Add T1 theorem**

Add:

```lean
theorem t1_i_eq_one {n j : ℕ} (hj : 2 ≤ j) (hjn : 2 * j ≤ n) :
    ∃ p : ℕ, commonPrimeDivisor n 1 j p := by
  ...
```

Proof sketch:
- Assume no such prime.
- Use `Nat.coprime_of_dvd` to show `n.Coprime (Nat.choose n j)`.
- Use `Nat.add_one_mul_choose_eq (n - 1) (j - 1)` to get `n ∣ j * Nat.choose n j`.
- Use `Nat.Coprime.dvd_mul_right` to derive `n ∣ j`.
- From `0 < j`, get `n ≤ j`; contradiction with `2*j ≤ n`.
